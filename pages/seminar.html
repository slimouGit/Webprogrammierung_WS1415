Web-Programmierung

Weiterführende Informationen zu interessanten Themen des World Wide Webs.

  • HTML

     HTML bedeutet HyperText Markup Language. Markup Language bedeutet, dass es eine Auszeichnungssprache ist, mit der Strukturen in einem Text geschaffen werden.
    • semantisch korrrektes HTML

  • CSS

     CSS (Cascading Style Sheets). Wer Webseiten erstellen möchte, wird sich intensiv mit CSS beschäftigen müssen und wollen. Sie sind für die Layoutgestaltung elementar geworden, gerade bei der Erstellung von großen Websites.

  • JavaScript

     Über JavaScript haben Sie mit Sicherheit schon Einiges gehört oder gelesen, darunter sowohl positives, als auch negatives. Immer wieder gibt es Meldungen darüber, wie unsicher JavaScript sei, viele Nutzer blocken JavaScript in ihrem Browser sogar vollständig. Sicherlich ist es so, dass JavaScript Schaden anrichten kann – zumal es vom Nutzer weitestgehend unbemerkt bleibt. Doch den eigentlichen Schaden richtet dabei der Programmierer der Scripts an, der unerlaubt Sourcecodes einschleust und damit z. B. Daten des Besuchers ausliest.

  • Validierung

     Das Validieren ist eine Prüftätigkeit bei der Produkte, Protokolle oder Dokumente in Bezug auf ihre Spezifikationen geprüft werden. (Quelle: http://www.html-seminar.de/)
    • diese Website validieren (validator.w3.org)

  • Webstandards

     Das Layout ist fertig, der Text lässt auf sich warten.

  • Accessibility

     edingt durch den hohen Stellenwert, den das Internet für die Menschen in der heutigen Informationsgesellschaft hat, ist es wichtig, dass niemandem der Zugang zu diesem Medium verwehrt wird. Leider existieren im Web jedoch viel zu viele Angebote, die aus unterschiedlichen Gründen von Personen mit situationsbedingten Einschränkungen, Menschen mit Behinderungen und Senioren nicht genutzt werden können. (Quelle: Modul Mediendesign II, Einführung in die Barrierefreiheit)

  • Usability

     Das englische Wort „Usability“, das auch im deutschen Sprachraum genutzt wird, lässt sich am Besten mit „Gebrauchstauglichkeit“ übersetzen. Der Begriff „Gebrauchstauglichkeit“ wurde 1999 gewählt, als die Norm [DIN9241-11] in das Deutsche übersetzt wurde. Diese beiden Begriffe sind also als synonym zu betrachten.
    • Usability Engineering
    • Usability Evaluation
    (Quelle: Mensch Computer Kommunikation (MCK) | Prof. Dr. Jörg Thomaschewski)

  • XML

     ML ist das Akronym, also die Abkürzung für eXtensible Markup Language, einer Auszeichnungssprache, mit der Webseiten strukturiert werden können. Dieses Kapitel lehrt Sie den Umgang mit XML, Ziel ist es, zu verstehen, wozu XML dient.
